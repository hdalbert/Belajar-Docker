Lab 6. Whats Going on in Containers
- CLI monitoring & inspection

	- docker container top - process list in one container
		- top command to list all process that will be containing in a singgle running container

	- docker container inspect - details of one container config
		- show metadata about the container (startup, config, volumes, networking, etc)

	- docker container stats - container stats for all container 
		- show live performance data for all container
